# Kwik Eats Admin

Demo admin panel built with Next.js 14 (App Router), TypeScript, TailwindCSS, Framer Motion, Recharts, NextAuth v4.

## Quick start
1. Ensure Node.js 18+ is installed.
2. unzip and cd into folder
3. npm install
4. npm run dev
5. Open http://localhost:3000

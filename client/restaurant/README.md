# Restaurant (admin) app

This is the Restaurant Manager (admin) Flutter application.

Quick start (PowerShell):

```powershell
cd client/restaurant
flutter pub get
flutter run -d chrome
```

If you want to build an Android APK:

```powershell
flutter build apk
```

Project structure highlights:

- `lib/` — app source code (screens, widgets, providers)
- `android/`, `ios/`, `web/`, `linux/`, `macos/`, `windows/` — platform folders
- `pubspec.yaml` — dependencies

See the root `README.md` for monorepo-level commands.

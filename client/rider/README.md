# Rider app (placeholder)

This folder is reserved for the Rider Flutter app.

To scaffold a new Rider app here:

```powershell
cd client
flutter create rider
cd rider
flutter pub get
```

Common Rider features to implement:
- View assigned orders
- Navigate to pickup/dropoff locations
- Update order status
- Profile (vehicle selection)

You can copy shared code (widgets/services) from `client/restaurant/lib/` as appropriate.
